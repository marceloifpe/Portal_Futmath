Sobre Este Projeto

O Portal Brasileirão foi desenvolvido como projeto da disciplina de Desenvolvimento Web. O objetivo é criar uma plataforma estilo Transfermarkt focada nos times das Séries A e B do Campeonato Brasileiro.

  • CSS3
  • JavaScript (Vanilla)
  • API football-data.org
