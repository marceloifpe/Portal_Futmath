Sobre o Projeto

O que é o Portal Futmath?

O Portal Futmath é um projeto web desenvolvido para a disciplina de Desenvolvimento Web, ministrada pelo Professor Fabio Feliciano. O objetivo do projeto é criar uma plataforma estilo Transfermarkt focada exclusivamente no desempenho dos times na liga nacional de seu país, onde nessa versão tem a liga brasileira e inglesa.

O portal permite que os usuários explorem informações detalhadas sobre os times, visualizem elencos completos, acompanhem históricos de partidas, conheçam as conquistas dos clubes e muito mais.

Funcionalidades Principais

🏆

Seleção de Times

Navegue entre todos os times da Série A do Brasileiro e da Premier League da Inglaterra

👥

Elencos Completos

Visualize todos os jogadores, posições e informações detalhadas

📊

Estatísticas

Acompanhe estatísticas gerais, posição na tabela e artilheiros

⚽

Histórico de Partidas

Confira últimas e próximas partidas do time

🎖️

Conquistas

Explore títulos e troféus conquistados pelos clubes

📱

Design Responsivo

Acesse o portal em qualquer dispositivo (mobile, tablet, desktop)

Tecnologias Utilizadas

HTML5

Estrutura semântica e acessível do projeto, com tags como <header>, <nav>, <main>, <section>, <article> e <footer>

CSS3

Design responsivo com media queries, variáveis CSS, flexbox, grid, transições e animações para uma experiência visual moderna

JavaScript (Vanilla)

Lógica de navegação, manipulação do DOM, requisições assíncronas com Fetch API e gerenciamento de dados com localStorage

API football-data.org

API gratuita que fornece dados em tempo real sobre times, jogadores, partidas e estatísticas do Brasileirão

Recursos de Acessibilidade

  • Atributos Alt: Todas as imagens possuem descrições alternativas para leitores de tela
  • Navegação por Teclado: Todos os elementos interativos são acessíveis via teclado
  • Semântica HTML: Uso correto de tags semânticas para melhor estrutura e acessibilidade
  • Contraste de Cores: Cores escolhidas com contraste adequado para melhor legibilidade
  • Responsividade: Design adaptável para diferentes tamanhos de tela
  • Aria Labels: Uso de atributos ARIA para melhor comunicação com tecnologias assistivas

Estrutura do Projeto

O projeto é organizado da seguinte forma:

brasileirao-portal/
├── index.html              (Página inicial)
├── style.css               (Estilos globais)
├── script.js               (Lógica principal)
├── paginas/
│   ├── time.html           (Detalhes do time)
│   ├── elenco.html         (Lista de jogadores)
│   ├── jogador.html        (Detalhes do jogador)
│   ├── historico.html      (Histórico de clubes)
│   ├── partidas.html       (Partidas do time)
│   ├── conquistas.html     (Títulos e troféus)
│   ├── estatisticas.html   (Dados gerais)
│   ├── sobre.html          (Sobre o projeto)
│   └── contato.html        (Informações de contato)
└── Imagens/
    ├── logo.png
    ├── favicon.ico
    └── times/              (Escudos dos times)
